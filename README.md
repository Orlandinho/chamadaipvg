## Sobre o projeto

Apenas uma simples lista de chamada para facilitar a organização da Escola Dominical da Igreja onde congrego. Abaixo eu listarei todas as dependências que estou usando ao longo do desenvolvimento desse projeto

- [Tailwind CSS](https://tailwindcss.com).
- [Inertia](https://inertia.com).
- [Vue 3](https://vue.org).
- [Laravel](https://laravel.com).
- [HeroIcons](https://heroicons.com).
- [HeadlessUI](https://headlessui.com).
- [prettier](https://prettier.io).
- [Maska](https://beholdr.github.io/maska/#/).
